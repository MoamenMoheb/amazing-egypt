### 1. Frontend (Interactive App)
This is the main application where you can explore the map and sites.

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies (if not already done):
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser at: [http://localhost:5173](http://localhost:5173)

### 2. Backend (API)
The backend service handles data and admin panel functionality (currently basic).

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Run the Go server:
   ```bash
   go run cmd/api/main.go
   ```
3. API runs at: [http://localhost:8080](http://localhost:8080)
